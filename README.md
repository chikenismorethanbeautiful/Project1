# 高考王 - 高考志愿填报助手

## 项目简介

「高考王」是一款基于大语言模型的高考志愿填报智能咨询系统。系统模拟张雪峰老师的语言风格和底层逻辑，为考生提供个性化、数据驱动的志愿填报咨询服务。

## 核心功能

### 用户端

- **用户认证**：支持用户名密码注册/登录、手机验证码登录
- **对话管理**：创建/切换/删除对话，自动生成对话标题
- **智能问答**：模拟张雪峰风格回答高考相关问题
- **工具调用**：
  - 大学信息搜索
  - 录取分数线查询（含年份自动回退）
  - 大学排名检索
  - 知识库语义检索
- **语音合成**：AI回复一键朗读，支持缓存与全局静音
- **图片识别**：上传含文字的图片，OCR识别后附加至消息
- **缅怀纪念**：特定关键词触发纪念活动模块（计时、献花）

### 管理端（后台）

- 数据概览（用户/对话/消息统计、近7天趋势、活跃用户排行）
- 用户管理（查看、搜索、删除）
- 对话与消息记录管理
- 词频统计（用户消息分词 + 词云展示）
- 操作日志（管理员操作记录）
- Token消耗统计（近7天趋势）

## 技术架构

| 层级       | 技术选型                                    |
| ---------- | ------------------------------------------- |
| 后端框架   | Django (MTV模式)                            |
| 数据库     | MySQL                                       |
| 向量数据库 | FAISS                                       |
| 嵌入模型   | BAAI/bge-large-zh-v1.5                      |
| 大模型     | 阿里云通义千问 (DashScope)                  |
| 语音合成   | 讯飞 TTS (WebSocket)                        |
| 图片识别   | OCR.Space API                               |
| 前端       | 原生 HTML/CSS/JS + marked.js + highlight.js |
| 部署       | Nginx + uWSGI / Gunicorn                    |

## 系统流程

```
用户输入 → 大模型识别意图
    ├── 知识库检索（张雪峰观点）
    ├── 工具调用（大学/分数线/排名）
    └── 生成回复 → 持久化 → 前端渲染
```

## 工具调用说明

系统为大模型配置了五个可调用的工具函数：

| 工具名称              | 功能说明                       |
| --------------------- | ------------------------------ |
| `search_skills`       | 检索张雪峰观点知识库           |
| `search_university`   | 查询大学基本信息               |
| `get_admission_score` | 查询录取分数线（自动年份回退） |
| `get_rank`            | 获取软科排名                   |
| `get_current_time`    | 获取当前系统时间               |

## 知识库构建

- 数据来源：张雪峰历年讲座/文章（Markdown格式）
- 分块策略：`chunk_size=500`, `chunk_overlap=50`
- 向量化：BAAI/bge-large-zh-v1.5 → 768维向量
- 索引存储：FAISS（本地持久化）
- 查询方式：语义相似度检索，返回 Top-3 相关片段

## 数据库设计（核心表）

| 表名                     | 说明                               |
| ------------------------ | ---------------------------------- |
| `chat_user`              | 用户信息（昵称、头像、静音偏好等） |
| `chat_conversation`      | 对话会话（session_id、标题、时间） |
| `chat_message`           | 消息记录（角色、内容、时间）       |
| `chat_verification_code` | 手机验证码（类型、过期、使用状态） |

- 讯飞 TTS API Key

## 快速启动（简要）

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 配置数据库与API密钥（settings.py / .env）

# 3. 迁移数据库
python manage.py makemigrations
python manage.py migrate

# 4. 构建知识库（首次启动自动构建）
python manage.py shell -c "from chat.services import init_vectorstore; init_vectorstore()"

# 5. 启动服务
python manage.py runserver
```

> 生产环境建议使用 Nginx + uWSGI 部署。

## 不足与后续改进方向

- 大模型响应耗时仍偏高（p95 ≈ 17.6秒），可引入流式输出与缓存
- 第三方API（分数线）存在超时风险，计划自建本地数据库
- 知识库内容有待持续扩充
- 语音合成暂不支持断点续播与语速调节
- Token统计基于估算，待接入官方准确用量
